# v1.3 NLP Improvements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the NLP gap — take conversational prompt injection from 10% to 80%+ with zero false positive increase.

**Architecture:** Add an inline `def preprocess_prompt` to match.jq that extracts intent from natural language via 9 pattern matchers, outputs a metadata sidecar consumed by scoring. Fix case-insensitive index bug, shell_command "make" false positive, and add safety heuristics.

**Tech Stack:** Pure bash + jq (ONIG regex engine). No new dependencies.

**Spec:** `docs/superpowers/specs/2026-03-27-v1.3-nlp-improvements-design.md`

def keyword_candidates:
